
Api-Watchman is a open-source lightweight terminal based api client

commands
  set : sets variables { set key value }
  rm : remove variables { rm key }
  log : prints variables { log key }
  clear : clears the console { clear }
  opt : change fetch options { opt key value}
  header : change headers { header key value }
  load : load .env variable { load var_name object key }
  exit / quit : exit Api-watchman

enter variable_name or url to fetch
vaddriables are stored in ./config.json
fetch options are stored in ./options.json

CLI options
  -p change watch path
  -l print variables
  -h help

config.json contains
  def : name of the working url (fetched on file change)
  delay : ms to sleep before fetching url
  type : middle ware to be applied [json,blob,text]
